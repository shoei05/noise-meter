# 騒音計

スマホで使える騒音計ウェブアプリ。

## 機能

- リアルタイムdB測定
- 10秒間の平均・最小・最大値表示
- グラフ表示
- PWA対応（ホーム画面に追加可能）

## デモ

https://shoei05.github.io/noise-meter/

## 技術

- React + TypeScript
- Vite
- Tailwind CSS
- Chart.js
- Web Audio API

## 開発

```bash
npm install
npm run dev
```

## 注意

- マイクへのアクセス許可が必要
- 測定値は参考値（機種により精度が異なる）
